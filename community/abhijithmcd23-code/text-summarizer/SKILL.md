# Text Summarizer Skill

## Purpose
Summarize long documents.

## When To Use
- Articles
- Research papers
- Notes

## Instructions
1. Read full input
2. Find key ideas
3. Generate summary
4. Output key points

Rules:
- Preserve meaning
- Remove repetition
- Be concise
